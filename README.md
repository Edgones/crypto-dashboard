## 📝 Sobre o Projeto

Este é um Dashboard Financeiro desenvolvido para monitoramento em tempo real do valor do Bitcoin. O projeto foi construído focando em **fundamentos sólidos de Engenharia de Software**, consumo de APIs REST e experiência do usuário (UX) com foco em performance.

## 🛠️ Tecnologias e Conceitos Aplicados

- **JavaScript Assíncrono (Async/Await):** Comunicação eficiente com a API sem travar a interface.
- **API REST (CoinGecko):** Integração com dados reais de mercado utilizando autenticação via API Key.
- **Manipulação de DOM:** Injeção dinâmica de dados financeiros no HTML.
- **Internacionalização (Intl):** Formatação de moeda no padrão brasileiro (ponto no milhar e vírgula no decimal).
- **CSS Grid & Variables:** Layout responsivo (Mobile-First) e design system escalável.
- **Git/GitHub:** Controle de versão profissional seguindo o padrão de _Conventional Commits_.

## 📊 Funcionalidades

- [x] Cotação em tempo real (BTC/USD e BTC/BRL).
- [x] Formatação financeira de precisão (66.408,3).
- [x] Atualização automática via Polling (60 segundos).
- [x] Tratamento de erros resiliente (Failsafe).
